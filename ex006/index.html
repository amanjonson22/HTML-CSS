Lista Ordenada

Aqui veremos um pouco sobre listas

A lista <ol> significa Ordered List, são listas numeradas.
Para listar itens, dentro do OL, usa-se <li>.
Elas contém alguns tipos de numeração: a; A; 1; I; e i.
I e i - são números romanos.

Lista Não Ordenada

A lista não ordenada contém pontos, quadrados e círculo vazio.
Utiliza-se o <ul> para a lista. E type para selecionar, podendo ser:
disc - círculo pintado(padrão)
circle - círculo vazio

Lista de Definição

É uma lista para definições. Para definir itens. Usa-se <dl> para colocar a lista.
<dt> para definir o termo a ser explicado.
<dd> para colocar a definição do termo.
