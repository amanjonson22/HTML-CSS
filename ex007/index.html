HYPERLINKS

Agora vamos aprender a colocar hyperlinks.

O que são?

Permitem que o usuário ligue um ponto a outro na internet. Seja acessando sites ou baixando arquivos, o usuário está interagindo com hyperlinks.

Usando hyperlinks

Para usar um hyperlink, usa-se <a>. O principal atributo dessa tag é o href

Acesse meu perfil do github

Outro atributo útil é o hreflang, que indica qual o idioma principal do site para onde o link redireciona. Isso permite avisar o navegador e softwares como lidar caso o visitante opte por traduzir os conteúdos.

Site da Universidade de Waterloo (em inglês)

Por padrão, quando se clica em um hyperlink, o site é aberto na mesma página que o usuário estava, ou seja, o conteúdo vai deixar de ser exibido.
É um comportamento desejado quando o visitante se mantém no site, apenas mudando de um doc para outro. Para controlar para onde ele vai, pode-se usar o atributo target, que suporta os seguintes valores:

  • _blank - abre em uma nova janela em branco
  • _self - abre na página atual (padrão)
  • _top - desfaz todos os frames e abre no navegador completo
  • _parent - similar ao _top, faz referência à uma janela mãe.
  • nome-do-frame - caso esteja usando frames, indica o nome da janela a abrir.

Downloads

Basta fazer o link diretamente para o arquivo que se deseja efetuar o download e adicionar o atributo download com o valor configurado para o nome do arquivo a ser baixado e o atributo para indicar ao navegador que tipo de arquivo está sendo baixado.
